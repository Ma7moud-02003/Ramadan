# Quran Memorization App 📖

A **Front-End Angular project** that helps users memorize the Quran with personalized plans and audio recitations. All user data and progress are stored using **Firebase**.

## Features ✨
- **User Authentication**: Users can sign up and log in using Firebase Authentication.
- **Personalized Memorization Plan**: Users can create their own memorization plan by selecting the verses they want to memorize.
- **Daily Verses**: Every day, the app shows the verses chosen by the user.
- **Audio Recitations**: Users can listen to the verses in the voice of the selected reciter (Sheikh of choice).
- **Progress Tracking**: Keep track of memorization and daily recordings.

## Technologies 🛠️

- **Frontend**: Angular, TypeScript, HTML5, CSS3, Tailwind CSS
- **Backend / Database**: Firebase (Authentication & Firestore)
- **State Management**: RxJS, Angular Signals
